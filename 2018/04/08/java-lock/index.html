锁简介

我们通常说的锁，就是线程锁，也就是加上锁后对当前线程有效。为什么是针对线程的呢，同一线程顺序执行也就不需要锁好吧。
锁是通过对同一内存的使用产生竞争来达到同一时间只能一个线程对其访问。
