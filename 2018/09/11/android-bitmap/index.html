使用合适的 Config

在把资源解析为 Bitmap 过程中通过设置 options.inPreferredConfig, 或者在创建 Bitmap (Bitmap.createBitmap) 时指定 Config, Config 的值有以下几种：

ALPHA_8:无颜色，只有透明度
RGB_565:只有颜色，无透明度
ARGB_4444:一个像素占8个字节
ARGB_8888:一个像素占4个字节，相比 ARGB_4444, ARGB_8888 更推荐使用

根据自己需要选择使用
